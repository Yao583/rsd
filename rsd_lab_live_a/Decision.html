This experiment consists of two parts. Each part has {{ nr_rounds }} periods. Once you finish the study one period is randomly selected and your bonus payment corresponds to the earnings you receive in that randomly selected period.

{% trans "Part 1" %}

{% blocktrans trimmed %} In this study, you and {{ nr_others }} other participants are going to play a game for {{ nr_prizes }} prizes. Each prize is worth money, but might be worth a different amount of money for each participant. {% endblocktrans %}

{% blocktrans trimmed %} You and the other participants will each rank prizes in any order you wish. Then, an allocation process will use these rankings to allocate the prizes, one for each participant. {% endblocktrans %}

{% blocktrans trimmed %} The allocation process attempts to give each participant a prize that they ranked higher rather than a prize that they ranked lower. However, this is not always possible, since the allocation process must take into account the rankings of all participants. {% endblocktrans %}

{% trans "Step 3: Allocation Process" %}

{% blocktrans trimmed %} In Step 3, the allocation process allocates the prizes to participants. {% endblocktrans %}

{% trans "How the allocation process works:" %}

{% blocktrans trimmed %} These details are important to learn: You may be able to apply your knowledge of them to make better decisions in deciding how many prizes you want to rank and in what order. {% endblocktrans %}

{% blocktrans trimmed %} Some details may seem confusing at first. This is quite natural! But don't worry, we will show you step-by-step examples. Things will become clearer along the way. {% endblocktrans %}

{% trans "Overview of the allocation process" %}

{% blocktrans trimmed %} First, the process aims to match participants with prizes based on their rankings. {% endblocktrans %}

{% blocktrans trimmed %} The first part of the allocation process works in the following way: {% endblocktrans %}

{% blocktrans trimmed %} Round 1: The participant with the first prize priority goes first. That participant will be assigned the prize they submitted first in their ranking. If this participant ranked no prizes, they remain unmatched and receive no prize. {% endblocktrans %}

{% blocktrans trimmed %} If a prize was assigned to the participant with the first prize priority, that prize is excluded from the set of available prizes. {% endblocktrans %}

{% blocktrans trimmed %} Round 2: The participant with the second prize priority goes second. That participant will be assigned the top prize according to the ranking they submitted among the set of available prizes. If the participant has not ranked any of the available prizes, they remain unmatched and receive no prize. {% endblocktrans %}

{% blocktrans trimmed %} The set of available prizes is adjusted to exclude any prizes assigned to participants in Rounds 1 and 2. {% endblocktrans %}

{% blocktrans trimmed %} And so on. {% endblocktrans %}

{% blocktrans trimmed %} In general, in Round X: The participant with the Xth prize priority goes Xth. That participant will be assigned the top prize according to the ranking they submitted among the set of available prizes. If the participant has not ranked any of the available prizes, they remain unmatched and receive no prize. {% endblocktrans %}

{% blocktrans trimmed %} At the end of this first part of the allocation process, some participants have received prizes and some participants may be unmatched. Likewise, some prizes were allocated to participants, and some prizes may remain available prizes. {% endblocktrans %}

{% blocktrans trimmed %} Every unmatched participant, and only those, participate in the second allocation process, and all prizes that are still available, and only those, participate in the second allocation process. {% endblocktrans %}

{% blocktrans trimmed %} The second part of the allocation process works as follows. Every participant who is unmatched randomly receives one of the prizes that are still available. In this second allocation process, your ranking of prizes is not taken into account. At the end of the second allocation process, every previously unmatched participant is matched to a prize, whether they ranked it or not. {% endblocktrans %}

{% blocktrans trimmed %} At the end of Step 3, that is after the first and the second part of the allocation process, every participant is therefore matched to a prize which determines their earnings. {% endblocktrans %}
